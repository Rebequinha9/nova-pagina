Minha pesquisa sobre html e css

Oque é html?

HTML é uma (linguagem de marcação de hipertexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web, ele serve para organizar o conteúdo de uma página com textos linhas e imagensz

Oque é css?

CSS é uma linguagem de estilo para a definição da apresentação visual, enquanto o HTML é uma marcação que define uma estrutura de uma página

Oque é responsividade?

Responsividade É a capacidade de um site se ajustar automaticamente e a qualquer tamanho de tela, garantindo uma navegação e visualização mais ampla para qualquer dispositivo e qualquer pessoa
